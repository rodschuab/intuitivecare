# Teste 1 – Processamento de Dados da ANS

## Objetivo
Este projeto tem como objetivo realizar o download automático dos arquivos disponibilizados pela ANS, 
extrair os dados de despesas dos últimos trimestres disponíveis, consolidar essas informações em um único arquivo CSV e 
gerar um arquivo compactado com o resultado final.


---

## Tecnologias Utilizadas
- Java 17
- Maven
- OpenCSV
- IntelliJ IDEA

---

## Como Executar o Projeto

### Pré-requisitos
- Java 17 instalado
- Maven configurado

### Execução
1. Clone o repositório
2. Acesse a pasta `teste1`
3. Execute a classe `Main.java`

Ao executar, o sistema realiza automaticamente:
- Download dos arquivos ZIP da ANS
- Extração dos arquivos CSV
- Leitura e processamento dos dados
- Geração do arquivo consolidado
- Compactação do resultado final
